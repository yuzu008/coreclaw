# 教材作成スキル

> 教授設計原則に基づいた、効果的な教材（ワークシート、スライド、クイズ等）を生成します。

## 概要

このスキルは、メタプロンプトで収集したコンテキストを基に、学習目標に適した教材を作成します。Gagné's Nine Events of InstructionやARCS Modelを適用し、学習効果を最大化します。

## 入力要件

メタプロンプトから以下の情報を受け取ります：

- **必須**: 学年、教科、単元/トピック、教材形式
- **推奨**: 学習目標、難易度、時間配分
- **任意**: デザイン要件、差別化対応

## 学習指導要領参照

> **小中高の教材作成時は、必ず学習指導要領を参照すること。**
>
> **重要**: 学習指導要領の検索には `data/curriculum.db` (SQLite) を使用してください。
> `data/curriculum/*.md` を grep で検索しないでください（遅いため）。

### 検索方法

```bash
# 教科・単元のキーワードで検索（3文字以上）
sqlite3 data/curriculum.db "SELECT school_level, heading, substr(body,1,500) FROM sections_fts f JOIN sections s ON f.rowid = s.rowid WHERE sections_fts MATCH '{教科名や単元の3文字以上のキーワード}' AND s.school_level = '{学校種}' LIMIT 5;"

# 2文字キーワードの場合
sqlite3 data/curriculum.db "SELECT school_level, heading, substr(body,1,500) FROM sections WHERE school_level = '{学校種}' AND (heading LIKE '%{キーワード}%' OR body LIKE '%{キーワード}%') LIMIT 5;"
```

教材には必ず以下のセクションを含めます：

```markdown
### 学習指導要領との関連

**参照箇所**: {学校種}学習指導要領 第{章}章 第{節}節 {教科} {学年}
**目標**: 「{学習指導要領の目標を引用}」
**内容**: 「{当該単元の内容を引用}」
**本教材との対応**: {教材内容が学習指導要領のどの部分に対応するか}
```

## 対応する教材形式

| 形式 | 説明 | 適した場面 |
|------|------|-----------|
| `worksheet` | ワークシート・プリント | 演習、確認、宿題 |
| `slide` | プレゼンテーションスライド | 説明、講義、まとめ |
| `quiz` | クイズ・小テスト | 形成的評価、復習 |
| `handout` | 配布資料・参考資料 | 補足説明、自習用 |

## 教授設計原則の適用

### Gagné's Nine Events of Instruction

教材設計時に以下の9事象を考慮：

1. **注意の獲得** - 興味を引く導入
2. **目標の提示** - 学習目標の明示
3. **前提条件の想起** - 既習事項の確認
4. **学習内容の提示** - 新しい情報の提示
5. **学習の案内** - ガイダンス・ヒント
6. **練習の誘発** - 演習機会の提供
7. **フィードバックの提供** - 正誤の確認
8. **評価の実施** - 習得度の確認
9. **保持と転移の促進** - 応用・発展

### ARCS Model

動機づけの4要素を教材に組み込み：

- **A (Attention)**: 興味を引く要素
- **R (Relevance)**: 学習者との関連性
- **C (Confidence)**: 達成可能な難易度
- **S (Satisfaction)**: 達成感を得られる構成

## 差別化対応（UDL原則）

Universal Design for Learningに基づき、多様な学習者に対応：

### 提示の多様化（Multiple Means of Representation）
- テキスト + 図解 + 音声説明
- 複数の具体例
- 用語の定義を明示

### 表現の多様化（Multiple Means of Action & Expression）
- 選択式 + 記述式の組み合わせ
- 図や絵で回答可能な問題
- 段階的なヒント

### 取り組みの多様化（Multiple Means of Engagement）
- 興味関心に応じた例題
- 選択可能な発展課題
- 自己評価セクション

## 主要な教材設計理論

| 理論 | 用途 |
|------|------|
| Gagné's Nine Events | 9つの教授事象に基づく教材構成 |
| ARCS Model | 動機づけを高める教材設計 |
| Universal Design for Learning | 多様な学習者への対応 |
