# 個別指導計画スキル

> 学習者の特性に応じた、個別化された指導戦略と支援計画を生成します。

## 概要

このスキルは、学習者プロファイルを基に、Zone of Proximal Development（最近接発達領域）理論などを適用し、個々の学習者に最適化された指導計画を作成します。

## 入力要件

メタプロンプトから以下の情報を受け取ります：

- **必須**: 学習者の状況（学年、教科、課題）
- **推奨**: 学習スタイル、習熟度、特別な配慮事項
- **任意**: 強み、興味関心、過去の指導履歴

## 学習指導要領参照

> **小中高の個別指導計画作成時は、必ず学習指導要領を参照すること。**
>
> **重要**: 学習指導要領の検索には `data/curriculum.db` (SQLite) を使用してください。
> `data/curriculum/*.md` を grep で検索しないでください（遅いため）。

### 検索方法

```bash
# 教科・単元のキーワードで検索（3文字以上）
sqlite3 data/curriculum.db "SELECT school_level, heading, substr(body,1,500) FROM sections_fts f JOIN sections s ON f.rowid = s.rowid WHERE sections_fts MATCH '{教科名や単元の3文字以上のキーワード}' AND s.school_level = '{学校種}' LIMIT 5;"

# 2文字キーワードの場合
sqlite3 data/curriculum.db "SELECT school_level, heading, substr(body,1,500) FROM sections WHERE school_level = '{学校種}' AND (heading LIKE '%{キーワード}%' OR body LIKE '%{キーワード}%') LIMIT 5;"
```

個別指導計画には必ず以下のセクションを含めます：

```markdown
### 学習指導要領との関連

**参照箇所**: {学校種}学習指導要領 第{章}章 第{節}節 {教科} {学年}
**目標**: 「{学習指導要領の目標を引用}」
**当該学年の内容**: 「{指導する内容の範囲を引用}」
**前学年との系統性**: {前学年での既習事項との関連}
```

## 学習スタイル対応

### 視覚型学習者
| アプローチ | 具体例 |
|-----------|--------|
| 図解・チャートの活用 | 概念マップ、フローチャート |
| 色分けによる整理 | 重要度別の色分け |
| 動画・アニメーション | 教育動画、シミュレーション |

### 聴覚型学習者
| アプローチ | 具体例 |
|-----------|--------|
| 口頭での説明強化 | 詳しい言語的説明 |
| ディスカッション | ペア・グループでの議論 |
| 音声教材 | ポッドキャスト、録音講義 |

### 運動感覚型学習者
| アプローチ | 具体例 |
|-----------|--------|
| 体験的学習 | 実験、フィールドワーク |
| 操作活動 | 具体物の操作、モデル作成 |
| 動きを伴う学習 | ロールプレイ、ゲーム |

### 読み書き型学習者
| アプローチ | 具体例 |
|-----------|--------|
| テキスト教材 | 詳細な説明文、参考書 |
| ノートテイキング | 要点整理、自分の言葉でまとめ |
| 書く活動 | レポート、振り返りジャーナル |

## 特別支援への配慮

### 注意事項

> ⚠️ **重要**: 特別支援が必要な学習者への対応については、以下の点にご注意ください：
> 
> 1. **専門家との連携**: 発達障害、学習障害等が疑われる場合は、必ず専門家に相談してください。
> 2. **診断は行わない**: AIは診断を行う立場にありません。提案する内容は一般的な支援策です。
> 3. **保護者との連携**: 特別な支援を行う場合は、保護者との十分な連携が重要です。

### 一般的な配慮例

| 困難さ | 配慮例 |
|--------|--------|
| 注意の持続 | 短い活動、視覚的なタイマー、明確な指示 |
| 読みの困難 | 音声読み上げ、行間・フォントサイズ調整 |
| 書きの困難 | 口頭回答、タイピング、穴埋め形式 |
| 記憶の課題 | 視覚的手がかり、チェックリスト、繰り返し |

## 主要な個別支援理論

| 理論 | 用途 |
|------|------|
| Zone of Proximal Development | 適切な支援レベルの設定 |
| Differentiated Instruction | 学習者に合わせた指導調整 |
| Universal Design for Learning | 多様な学習者への対応 |

## 次のステップ

個別指導計画作成後、以下の連携が可能です：

- `materials` → 学習者に適した教材を作成
- `feedback` → 進捗に対するフィードバックを生成
- `assessment` → 配慮事項を反映した評価を設計
