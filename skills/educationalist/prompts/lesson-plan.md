# 授業計画作成スキル

> Bloom's Taxonomyに基づいた、学習目標と整合性のある授業計画を生成します。

## 概要

このスキルは、メタプロンプトで収集したコンテキストを基に、学習目標を中心とした構造的な授業計画を作成します。Bloom's TaxonomyとGagné's Nine Events of Instructionを適用し、効果的な授業設計を支援します。

## 入力要件

メタプロンプトから以下の情報を受け取ります：

- **必須**: 学年、教科、単元/トピック
- **推奨**: 授業時間、学習目標、前提知識
- **任意**: 教室環境、特別な配慮、使用教材

## 学習指導要領参照

> **小中高の授業計画作成時は、必ず学習指導要領を参照すること。**
>
> **重要**: 学習指導要領の検索には `data/curriculum.db` (SQLite) を使用してください。
> `data/curriculum/*.md` を grep で検索しないでください（遅いため）。

### 検索方法

```bash
# 教科・単元のキーワードで検索（3文字以上）
sqlite3 data/curriculum.db "SELECT school_level, heading, substr(body,1,500) FROM sections_fts f JOIN sections s ON f.rowid = s.rowid WHERE sections_fts MATCH '{教科名や単元の3文字以上のキーワード}' AND s.school_level = '{学校種}' LIMIT 5;"

# 2文字キーワードの場合
sqlite3 data/curriculum.db "SELECT school_level, heading, substr(body,1,500) FROM sections WHERE school_level = '{学校種}' AND (heading LIKE '%{キーワード}%' OR body LIKE '%{キーワード}%') LIMIT 5;"
```

授業計画には必ず以下のセクションを含めます：

```markdown
### 学習指導要領との関連

**参照箇所**: {学校種}学習指導要領 第{章}章 第{節}節 {教科} {学年}
**目標**: 「{学習指導要領の目標を引用}」
**内容**: 「{当該単元の内容を引用}」
**本時との関連**: {本時の内容が学習指導要領のどの部分に対応するか}
```

## 学年段階別の調整

### 小学校低学年（1-2年）
- 具体的な活動・操作を中心に
- 1活動あたり10-15分で切り替え
- 視覚的な教材を多用
- ペア活動を取り入れる

### 小学校高学年（5-6年）
- 思考を促す発問を増やす
- グループ活動・話し合いを導入
- ノートへの記述を促す
- 既習事項との関連を意識

### 中学校
- 仮説→検証の流れを意識
- 個人思考→グループ→全体共有
- 言語活動の充実
- 教科横断的な視点

### 高等学校
- 探究的な活動を組み込む
- 主体的な学習設計
- ICTの効果的活用
- 社会との接続を意識

### 大学
- アクティブラーニング中心
- 反転授業の検討
- ルーブリック評価の明示
- 学術的な文脈での理解

## Bloom's Taxonomy 認知レベル

| レベル | 動詞例 | 活動例 |
|--------|--------|--------|
| 記憶 | 述べる、列挙する | 用語の確認、暗記 |
| 理解 | 説明する、比較する | 要約、言い換え |
| 応用 | 計算する、作成する | 問題解決、実践 |
| 分析 | 分析する、区別する | パターン発見、関係分析 |
| 評価 | 評価する、判断する | 批評、正当化 |
| 創造 | 設計する、提案する | 新しい解決策、作品制作 |

## Gagné's Nine Events の適用

1. **注意の獲得** → 導入での興味喚起
2. **目標の提示** → 学習目標の明示
3. **前提条件の想起** → 既習事項の確認
4. **学習内容の提示** → 新しい情報の提示
5. **学習の案内** → ガイダンス・ヒント
6. **練習の誘発** → 演習機会の提供
7. **フィードバック** → 正誤の確認
8. **評価の実施** → 習得度の確認
9. **保持と転移** → 応用・発展

## 次のステップ

授業計画作成後、以下の連携が可能です：

- `materials` → 授業で使用する教材を作成
- `assessment` → 授業の学習目標に対応した評価を設計
