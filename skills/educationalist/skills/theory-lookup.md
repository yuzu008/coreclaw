# 教育理論参照スキル

> 175件以上の教育理論データベースから、状況に適した理論を検索・参照します。

## 概要

このスキルは、教育実践における理論的裏付けを提供します。各スキルが生成するコンテンツに、エビデンスに基づいた教育理論の参照を付加します。

## 使用方法

他のスキルから理論を参照する場合：

1. **スキルの文脈を分析**: どの教育理論が適切かを判断
2. **理論の概要を取得**: 理論の説明と主要原則を確認
3. **適用方法を提案**: 具体的な実践への適用方法を記述
4. **出力に含める**: 生成コンテンツに「参照した教育理論」セクションを追加

### 出力フォーマット

```markdown
## 参照した教育理論

### {理論名}（{提唱者}）
**概要**: {理論の簡潔な説明}
**本実践への適用**: {どのように適用したか}
**主要原則**: {適用した主要原則}
```
