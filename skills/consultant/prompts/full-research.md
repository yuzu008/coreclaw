# 統合リサーチスキル（Full Research）

> 4フェーズを統合して実行する包括的リサーチワークフローです。

## 概要

Purpose Discovery → Deep Research → Framework Analysis → Report Writing の
4フェーズを一貫して実行します。各フェーズ間に品質ゲートを設け、分析品質を担保します。

## 実行フロー

```
Phase 1: Purpose Discovery
 │  ・構造化プロンプト（6要素） → ユーザー承認
 │  ・1問1答対話（最低3ラウンド）
 │  ・5 Whys + JTBD分析
 │  ・真の目的定義 → ユーザー承認
 │
 ├── 品質ゲート1: 真の目的定義済、計画作成済 ──┐
 │                                            │
Phase 2: Deep Research                        │
 │  ・Think→Action→Report サイクル              │
 │  ・日英並列検索                              │
 │  ・ソース信頼度評価                           │
 │  ・交差検証                                  │
 │                                            │
 ├── 品質ゲート2: 3+ソース、信頼度60%+ ──────────┤
 │                                            │
Phase 3: Framework Analysis                   │
 │  ・フレームワーク自動推薦                      │
 │  ・構造化分析                                │
 │  ・MECE検証                                 │
 │  ・ピラミッド構造化                           │
 │                                            │
 ├── 品質ゲート3: 1+フレームワーク適用 ───────────┤
 │                                            │
Phase 4: Report Writing                       │
 │  ・テンプレート選択                           │
 │  ・レポート生成                              │
 │  ・品質チェック（数値/引用/内容/形式）          │
 │  ・レポート提示 → ユーザー承認                 │
 │                                            │
 └── 完了 ─────────────────────────────────────┘
```

## 緊急度別ワークフロー

### 通常（デフォルト）
全フェーズを完全実行します。

### 急ぎ（「急ぎ」「今日中」「明日まで」）
- Phase 1: 質問を3問に限定
- Phase 2: 検索を3ラウンド以内に限定
- Phase 3: フレームワーク1つに限定
- Phase 4: エグゼクティブサマリー + 主要分析のみ

### 緊急（「今すぐ」「至急」「概要のみ」）
- Phase 1: 目的を直接確認（1問のみ）
- Phase 2: 検索を1ラウンドに限定
- Phase 3: スキップ（必要最低限の分析のみ）
- Phase 4: サマリーレポートのみ

## フェーズ間の連携

各フェーズの出力が次フェーズの入力となります。

| From | To | 引き渡しデータ |
|------|----|--------------|
| Phase 1 | Phase 2 | 真の目的、検索クエリリスト、調査範囲 |
| Phase 2 | Phase 3 | 収集データ、ソース一覧、信頼度評価 |
| Phase 3 | Phase 4 | 分析結果、インサイト、推奨アクション |

## ユーザー承認ポイント

以下のタイミングでは、**必ずユーザーの承認を待つ**こと。

1. **構造化プロンプト承認**（Phase 1 開始時）
2. **真の目的の承認**（Phase 1→2 移行時）
3. **レポート品質承認**（Phase 4 完了時）

承認なしに次のフェーズに進むことは禁止です。
