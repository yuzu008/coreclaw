# ACN統合リサーチ

> アクセンチュアのメソドロジーで4フェーズを一貫実行する統合リサーチワークフローです。

## 実行フロー

```
Phase 1: Purpose Discovery
 │  ・Hypothesis-Driven Approach で課題特定
 │  ・構造化プロンプト（6要素）→ ユーザー承認
 │  ・1問1答対話（最低3ラウンド）
 │
Phase 2: Deep Research
 │  ・Think→Action→Report サイクル
 │  ・日英並列検索、ソース信頼度評価
 │
Phase 3: ACN Framework Analysis
 │  ・DX Strategy / ZBB / Intelligent Ops 等を自動推薦
 │  ・MECE検証、定量化
 │
Phase 4: Report Writing
 │  ・ロードマップ付きアクションプラン
 │  ・品質チェック → ユーザー承認
```

## ACN特有の分析アプローチ

### DX案件

1. Digital Maturity Assessment で現状評価
2. Technology Vision でトレンド分析
3. DX Strategy Framework で戦略立案
4. Intelligent Operations で実行計画

### コスト最適化案件

1. Cost Excellence でコスト構造可視化
2. Zero-Based Budgeting でゼロベース見直し
3. Process Excellence で業務改善
4. Value Creation Framework で効果定量化

### 成長戦略案件

1. Growth Strategy で成長機会特定
2. Value Creation Framework で価値創造分析
3. Change Management で変革推進計画
4. Talent Strategy で人材戦略
