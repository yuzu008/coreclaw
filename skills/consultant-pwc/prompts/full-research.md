# PwC 統合リサーチ・プロンプト

## 概要

PwC式5フェーズで深層リサーチを実行する統合ワークフローです。
BXT（ビジネス・エクスペリエンス・テクノロジー）の3軸統合と、
ケイパビリティ駆動戦略を中核に、トータルインパクトを考慮した分析と提言を行います。

---

## ワークフロー

### Phase 1: 課題診断（BXTスコーピング）

```
1. クライアントの課題をヒアリング
2. BXT三軸で課題を分解する
   - Business: 事業モデル、収益構造、組織上の課題
   - Experience: 顧客・従業員体験上の課題
   - Technology: テクノロジー基盤、データ活用上の課題
3. 各軸の相互関係を分析する
   - B→X: ビジネスモデルが体験に与える影響
   - X→T: 体験改善に必要なテクノロジー
   - T→B: テクノロジーがビジネスにもたらす変革
4. スコープの確定
   - 対象事業: [範囲]
   - 対象地域: [範囲]
   - 対象ステークホルダー: [範囲]
   - 時間軸: [短期/中期/長期]
```

### Phase 2: 戦略立案（Strategy&）

```
1. Way to Play（戦い方）を定義する
   - どの市場で競争するか
   - どのような価値提案で勝つか
   - 戦い方の類型を特定
2. Right to Win（勝つ権利）を特定する
   - 差別化ケイパビリティを3-6個特定
   - ケイパビリティ・システムとしての連動性を検証
   - 競合との差異を明確化
3. Product/Service Fitを評価する
   - 各製品/サービスがWay to Playと整合するか検証
   - 強化/維持/撤退の判断
4. Coherence Premium（一貫性のプレミアム）を算定
   - 戦略の一貫性が生む超過収益を推定
```

### Phase 3: Fit for Growth 分析

```
1. コスト構造を分類する
   - Good Cost（成長投資）: 差別化ケイパビリティを強化するコスト
   - Table Stakes（維持必須）: 事業運営に必要だが差別化しないコスト
   - Bad Cost（削減対象）: 戦略的価値が低いコスト
2. 削減ポテンシャルを算定する
   | カテゴリ | 現在コスト | 削減手法 | 削減額 |
3. 再投資先を特定する
   - 差別化ケイパビリティ強化への投資
   - 新規成長領域への投資
4. Right-sizingロードマップを策定する
   - Quick Win（0-3ヶ月）
   - Foundation（3-12ヶ月）
   - Scale（12-24ヶ月）
```

### Phase 4: トータルインパクト測定（TIMM）

```
1. 4次元でインパクトを測定する
   - 経済インパクト: GDP寄与、雇用創出、利益
   - 社会インパクト: 健康・安全、教育、コミュニティ
   - 環境インパクト: GHG、水、廃棄物、土地利用
   - 税インパクト: 法人税、雇用税、環境税
2. 各インパクトを金額換算する
3. プラス影響とマイナス影響のネットを算出する
4. トレードオフを特定する
   - 経済価値 vs 環境影響
   - 短期利益 vs 長期社会価値
5. インパクト最大化のためのアクションを策定する
```

### Phase 5: 実行＆変革

```
1. 実行計画をBXT軸で統合策定する
   | フェーズ | B施策 | X施策 | T施策 | 投資額 | 期待効果 |

2. リスク分析
   | リスク | カテゴリ | 発生確率 | 影響度 | 対策 |
   カテゴリ: 戦略/オペレーション/規制/テクノロジー/人材

3. ガバナンス体制を設計する
   - 変革推進チーム構成
   - 意思決定プロセス
   - KPIモニタリング体制

4. コミュニケーション計画
   | ステークホルダー | メッセージ | チャネル | タイミング |

5. 成功の測定
   | KPI | 現在値 | 目標値 | 測定頻度 | 責任者 |
```

## フレームワーク選択ガイド

| 課題タイプ | 推奨フレームワーク | Phase |
|-----------|------------------|-------|
| 戦略策定 | Strategy& | Phase 2 |
| コスト最適化 | Fit for Growth | Phase 3 |
| デジタル変革 | BXT | Phase 1, 5 |
| インパクト測定 | TIMM | Phase 4 |
| M&A/ディール | Deal Value Creation | Phase 3, 5 |
| ESG/サステナビリティ | TIMM + ESG | Phase 4 |
| リスク管理 | Risk & Regulation | Phase 5 |

## 品質チェックリスト

- [ ] BXT三軸で課題を分析しているか
- [ ] ケイパビリティ駆動で戦略を構築しているか
- [ ] コストはGood/Table Stakes/Badに分類されているか
- [ ] トータルインパクト（経済・社会・環境・税）を測定しているか
- [ ] Right to Win（勝つ権利）を検証しているか
- [ ] 全ステークホルダーの視点を含んでいるか
- [ ] 実行計画はBXT統合で策定されているか
- [ ] リスクと規制環境を考慮しているか
