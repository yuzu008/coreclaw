# PwC フレームワーク分析プロンプト

## 概要

PwC固有のフレームワークを活用した分析テンプレート集です。
Strategy&、Fit for Growth、BXT、TIMMを中核に統合的な分析を行います。

## 使用ルール

1. **BXT統合**: ビジネス・体験・技術の3軸で常に検討する
2. **ケイパビリティ中心**: 「何ができるか」から戦略を構築する
3. **トータルインパクト**: 経済的価値だけでなく社会・環境・税への影響を測定
4. **Fit for Growth**: コスト削減だけでなく、成長への再投資を明示
5. **ステークホルダー全体**: 顧客・従業員・社会・株主すべての視点を含む
6. **Right to Win**: その市場で勝つ権利があるか常に問う
